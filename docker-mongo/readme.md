#### Install mongo and mongo-express containers
```bash
sudo docker pull mongo:latest  
sudo docker pull mongo-express  
```
localhost:8081  
#### From directory where docker-compose.yml is located:
sudo docker-compose up -d  
sudo docker-compose stop
